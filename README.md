# My Portfolio

This is my personal portfolio website built with **React**, **Vite**, **Bulma**, and **Highcharts**, showcasing my projects in **Data Analysis**, **Particle Collision Simulations**, **Web Development**, and **Artificial Intelligence**.

Live demo: [https://amilcartorres1000-web.github.io/portfolio/](https://amilcartorres1000-web.github.io/portfolio/)

---

## Features

- **React + Vite:** Fast and modern frontend development with Hot Module Replacement.
- **Bulma:** Responsive CSS framework for clean layouts.
- **Highcharts:** Interactive charts and real-time data visualizations.
- **Dynamic Graphs:**  
  - Live Data streaming for Data Analysis  
  - Particle Collision Simulation with realistic energy peaks  
  - Pie chart showing Web Development tech stack  
  - AI model training loss visualization  
- **GitHub Pages Deployment:** Hosted for free as a static site.
